(As a moderator, you might need help from the site administrators for making the change, but this document should still be useful to you so that you know what options are available.)

Your site might want to have some categories that are only available to a specific group of users. Codidact provides you with the following set of "trust level".

All trust levels can be set for category access (read) or for writing posts in the category (write). Reading and writing are not required to have the same trust level. However, setting a trust level doesn't allow users to do things they don't have the ability for.

Please note that trust levels are hierarchical, so that anyone who has one trust level has also all lower trust levels by default.

1. Everyone

As the name says, absolutely everyone has this trust level. This includes anonymous (unregistered) and new users.

Most write actions will require a registered user account, though, even if the trust level "everyone" has been selected.

2. Anyone with a user account

Requiring this trust level will exclude everyone (including search engine crawlers) that has no registered user account.

3. All but new users

This trust level requires users to have a registered account and some positive activity. They'll need to have earned the Participate Everywhere ability.

4. Veteran users

For the sake of this trust level, every user with either the Edit Posts or the Vote on Hold is considered "veteran".

5. Moderators only

Moderators, Admins, and the global variants thereof are the users that have this trust level.

6. Staff only

This trust level is only reached by users with the staff mark.
